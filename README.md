# AssingmentProgrammingGroupWeCyourC
Files containing assingment for group project

The program can be run in Visual Studio Community and Visual Studio Code

Task to do in order to run on:

**Visual Studio Code:**

1.Change your task json file

2.Change the args from "${file}",
  to
  "${fileDirname}\\main.c",
  "${fileDirname}\\functions.c",

**Visual Studio Community:**

1.Add all the necessary files into one folder

**All files that must be included:**

1.Stopwords.txt

2.ToxicWords.txt

3.read.txt

4.main.c

5.functions.c

6.Toxiclib.h
